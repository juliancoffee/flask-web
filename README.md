Lab work for Web Development at NTUU KPI

#Instalation
``` console
$ git clone https://github.com/juliancoffee/flask-web
$ cd flask-web
$ python -m venv venv
$ pip install requirements.txt
$ npm install
```

#Setup and run
``` console
$ cd src
$ source setup.sh
$ ./babel.sh &
$ flask run
```
then open localhost:5000 in your browser
